## Featured Project

### ZenkaiTV

ZenkaiTV is a modern anime hub for Web and Android TV.

- Multi-source catalogs and episode server picker.
- Japanese audio and Spanish subtitle preference.
- Embedded iframe player with direct-video safety separation.
- Android TV wrapper and local Windows server launcher.
- Vercel deployment with API routes.
