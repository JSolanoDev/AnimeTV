# FAQ

## Does AnimeTV host video files?

No. AnimeTV is a local app that displays metadata and connects to configured sources.

## Can English subtitles be translated to Spanish?

For direct text subtitle tracks, yes. For cross-origin iframe players, AnimeTV cannot rewrite the provider's internal subtitle track, but it can pass language preferences when the player supports them.

## Can I cast AniPub iframes?

Iframe embeds cannot be cast through the main video element. Use the iframe player's native cast option if the provider exposes one.
